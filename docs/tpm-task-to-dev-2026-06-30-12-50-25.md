# TPM Task: Implement Progressive Web App (PWA)

**Date/Time:** 2026-06-30 12:50:25
**Status:** Approved for Development
**Requested by:** User (Homeowner)
**Assigned to:** Dev Team

## 1. Executive Summary
Agar ART dapat "menginstall" aplikasi Kanban ini di layar utama HP Android mereka layaknya aplikasi native, kita perlu mengubah web app ini menjadi **Progressive Web App (PWA)**. Ini melibatkan penambahan `manifest.json`, Service Worker, dan ikon aplikasi.

## 2. Action Items: Frontend

### A. Web App Manifest
- [ ] Buat file `public/manifest.json`:
  ```json
  {
    "name": "Brodev Kanban Board",
    "short_name": "Brodev Kanban",
    "description": "Household Task Management Kanban Board",
    "start_url": "/",
    "display": "standalone",
    "background_color": "#1a1a2e",
    "theme_color": "#667eea",
    "orientation": "portrait",
    "icons": [
      { "src": "/icons/icon-192.png", "sizes": "192x192", "type": "image/png" },
      { "src": "/icons/icon-512.png", "sizes": "512x512", "type": "image/png" }
    ]
  }
  ```
- [ ] Tambahkan tag `<link rel="manifest" href="/manifest.json">` di `index.html`.
- [ ] Tambahkan meta tags untuk PWA di `index.html`:
  ```html
  <meta name="theme-color" content="#667eea">
  <meta name="apple-mobile-web-app-capable" content="yes">
  <meta name="apple-mobile-web-app-status-bar-style" content="black-translucent">
  <link rel="apple-touch-icon" href="/icons/icon-192.png">
  ```

### B. Service Worker
- [ ] Buat file `public/sw.js` dengan strategi **Network First** (agar data selalu fresh dari server, tapi tetap bisa cache shell UI):
  - Cache app shell (HTML, CSS, JS bundle).
  - Untuk request API, selalu gunakan network dulu. Jika offline, tampilkan halaman offline sederhana.
- [ ] Daftarkan service worker di `main.jsx` atau `index.html`:
  ```javascript
  if ('serviceWorker' in navigator) {
    window.addEventListener('load', () => {
      navigator.serviceWorker.register('/sw.js');
    });
  }
  ```

### C. App Icons
- [ ] Generate ikon aplikasi dengan ukuran:
  - `public/icons/icon-192.png` (192x192)
  - `public/icons/icon-512.png` (512x512)
  - Desain: Huruf "B" atau logo "Brodev" dengan background gradient ungu (#667eea → #764ba2).

### D. Offline Fallback Page (opsional tapi recommended)
- [ ] Buat `public/offline.html` — halaman sederhana yang muncul saat tidak ada koneksi internet, memberitahu user untuk reconnect.

## 3. Verification Plan
- Buka Chrome di Android → akses `http://[IP_LOKAL]:5173`
- Pastikan muncul prompt "Add to Home Screen" atau bisa diakses via menu Chrome > "Install app"
- Setelah "install", pastikan app berjalan **fullscreen** (tanpa address bar browser)
- Cek Lighthouse audit di Chrome DevTools → skor PWA harus hijau

## 4. Catatan TPM
Ini hanya perubahan di sisi Frontend. Backend tidak perlu diubah. Pastikan Vite di-konfigurasi agar file `manifest.json`, `sw.js`, `offline.html`, dan folder `icons/` ter-serve dengan benar dari `public/`.
