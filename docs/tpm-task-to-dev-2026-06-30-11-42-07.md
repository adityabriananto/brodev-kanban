# TPM Task: Implement Auth PIN Security for Owners

**Date/Time:** 2026-06-30 11:42:07
**Status:** Approved for Development (Execution)
**Requested by:** User (Homeowner)
**Assigned to:** Dev Team

## 1. Executive Summary
Ada temuan kritis dari sisi *Owner* terkait halaman Pemilihan Profil: ART saat ini dapat menekan profil Bapak/Ibu dan menyalahgunakan hak admin (memanipulasi tugas tanpa harus memasukkan alasan).
Kita akan menerapkan sistem keamanan **PIN 4-Digit** secara eksklusif untuk profil *Owner* untuk menutup *loophole* ini, sambil tetap membiarkan profil ART bisa masuk dengan satu klik (*frictionless*).

## 2. Action Items untuk Database
- [ ] Ubah tabel `users` (di `init.sql` dan database lokal yang sedang berjalan):
  ```sql
  ALTER TABLE users ADD COLUMN pin_code VARCHAR(4) NULL;
  ```
- [ ] Perbarui data awal (*seed data*):
  - Set `pin_code = '1234'` untuk pengguna dengan `role = 'owner'` (Bapak & Ibu).
  - Biarkan `pin_code = NULL` untuk pengguna dengan `role = 'art'`.

## 3. Action Items untuk Backend
- [ ] Buat *endpoint* baru: `POST /api/users/verify-pin`
  - *Payload*: `{ userId, pinCode }`
  - Lakukan kueri ke database untuk mencocokkan `pin_code` milik `userId` tersebut.
  - Kembalikan status HTTP `200` jika cocok, atau `401 Unauthorized` jika salah.

## 4. Action Items untuk Frontend (React)
- [ ] Modifikasi Halaman "Pilih Profil":
  - Jika pengguna menekan profil dengan `role === 'art'`, **langsung** arahkan (*redirect*) masuk ke *Board*.
  - Jika pengguna menekan profil dengan `role === 'owner'`, tampilkan **PinModal** (sebuah *overlay popup* bergaya *glassmorphism*).
- [ ] **Desain PinModal**:
  - Tampilkan 4 kotak input angka berukuran besar.
  - Fokus kursor otomatis berpindah ke kotak selanjutnya saat angka diketik.
  - Saat 4 angka terisi penuh, otomatis kirimkan *request* ke `POST /api/users/verify-pin`.
  - Jika salah, getarkan kotak (*shake animation*) dan kosongkan input.
  - Jika benar, izinkan masuk ke *Board* sebagai *Admin*.

## 5. Catatan TPM
Pastikan *hotfix* sekuriti ini diutamakan dalam *pipeline* kalian bersamaan dengan Socket.io. Pemilik Bisnis sangat menekankan agar PIN ini dirancang secara estetik dan responsif di layar sentuh HP (angka harus cukup besar untuk ditekan). Selamat *ngoding*!
